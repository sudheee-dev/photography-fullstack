# Role-Based Login — Angular

> Angular authentication app with route guards and role-based access control.

## ✨ Features
- Login & registration flow
- Role-based routing (admin / user)
- Angular Route Guards for protected pages
- Token storage & session management

## 🛠 Tech Stack
Angular` · `TypeScript` · `Angular Route Guards` · `RxJS`

## 🚀 Getting Started
bash
git clone https://github.com/sudheee-dev/Role-based-login-angular.git
cd Role-based-login-angular
npm install
ng serve
