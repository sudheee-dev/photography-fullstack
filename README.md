# 📸 Photography Full Stack Application

A full-stack photography sharing platform built using **Angular, Node.js, Express, and MySQL**. The application allows users to register, authenticate securely using JWT, upload photography posts, interact with other users, manage profiles, and perform role-based operations.

---

## 🚀 Features

### Authentication
- User Registration
- Secure Login
- JWT Authentication
- Password Encryption using bcrypt
- Protected Routes
- Logout

### User Profile
- View Profile
- Edit Profile
- Profile Picture
- Bio
- Followers List
- Following List

### Photography Posts
- Upload Photo
- Edit Own Posts
- Delete Own Posts
- View Feed
- Image Upload using Multer
- Location Support
- Description Support

### Social Features
- Like Posts
- Dislike Posts
- Follow Users
- Unfollow Users
- Followers
- Following

### Authorization
- User Role
- Admin Role
- Admin can edit/delete any post
- Users can edit/delete only their own posts

```

      │
      ▼
JWT Token Generated
      │
      ▼
Stored in Browser
      │
      ▼
Sent in Authorization Header
      │
      ▼
Backend Middleware
      │
      ▼
Protected APIs
```
